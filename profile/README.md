## ![Logo](https://totheos.com/favicon-16x16.png) [ToTheos](https://totheos.com) &nbsp;&nbsp; Async AI Engineering

Run autonomous AI agents as Tasks in the background to implement features, refactor repositories, generate tests, and open pull requests automatically.

Built for real software development workflows, not chat.

Prompts entered in the UI are executed as Tasks. Tasks are composed of Steps, and each Step executes a shell command. Optional auto-fix prompts can run automatically when a Step fails.
